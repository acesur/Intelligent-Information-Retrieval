How it Works

Web Crawler

Our system uses a specialized web crawler to extract publication data from Coventry University's PurePortal. The crawler collects information such as authors, publication years, titles, abstracts, and keywords.

Inverted Index

We build and maintain an efficient inverted index that enables fast and accurate searches. The index is automatically updated to include the latest publications.

Ranking Algorithm

Search results are ranked using the BM25 algorithm, a probabilistic ranking model that balances term frequency and document length to provide the most relevant results.

Regular Updates

The system is scheduled to automatically update its database weekly, ensuring that you always have access to the latest publications from Coventry University's School of Economics, Finance and Accounting.
